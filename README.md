•About Sweet Delights Cake Ordering System

-This is a comprehensive cake ordering system designed to provide a seamless experience for customers, staff, and administrators. The system allows customers to browse cakes, place orders, track their delivery, and view order history.

•System Features

-Customer Features: Browse menu, add items to cart, place orders, track orders, view order history, view cancel history
-Staff Features: View and update order statuses
-Admin Features: Manage users, manage menu items

#How to Use

•For Customers:

-Login with customer credentials
-Browse the cake menu and add items to your cart
-Proceed to checkout and select payment method (GCash or Cash on Delivery)
-Track your order status in real-time
-View your order history for past purchases
-View your cancel history for cancelled orders

•For Staff:

-Login with staff credentials
-View all orders in the system
-Update order status as they progress through preparation and delivery

•For Admin:

-Login with admin credentials
-Manage all users in the system
-Manage menu items (add, edit, delete)

#Payment Methods

•The system supports two payment methods:

-GCash: Digital payment through the GCash app
-Cash on Delivery: Pay with cash when your order is delivered

#Order Statuses

•Orders progress through the following statuses:

-Pending: Order received, awaiting processing
-Preparing: Order is being prepared
-Ready: Order is ready for delivery 
-Delivered: Order has been successfully delivered
-Cancelled: Order has been cancelled
